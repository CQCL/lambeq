Subpackages

Sub-package

Description

lambeq.ansatz

Concrete implementations of classical and quantum ansätze.

lambeq.ccg2discocat

Package containing the interfaces for the CCG parsers (including a

CCGBankParser), as well as the code for Bobcat parser.

lambeq.pregroups

A collection of useful utilities for easier manipulation of pregroup diagrams.

lambeq.reader

Abstractions and concrete classes for readers, implementing a variety of

compositional models for sentences.

lambeq.rewrite

Contains implementations of rewrite rules for the transformation of

string diagrams.

lambeq.tokeniser

Tokenisation classes and features for all parsers and readers.

lambeq.training

Provides a selection of trainers, models, and optimizers that greatly simplify

supervised training for most of lambeq’s use cases, classical and quantum.
